# Store Daily Notes as plain markdown source

Daily Notes are plain `.md` files, and the markdown text is the source of truth rather than a projection of a richer block, outline, JSON, or rich-text document model. This keeps notes readable and editable outside Jot, supports direct Google Drive recovery and manual repair, and makes sync conflicts visible in the file itself. The trade-off is that editor choices must be constrained by markdown round-trip behavior and source preservation rather than by the richest possible WYSIWYG editing model.
