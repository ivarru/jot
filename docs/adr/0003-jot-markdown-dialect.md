# Use CommonMark and GFM for Daily Notes

Jot Markdown is CommonMark with GitHub Flavored Markdown extensions, plus LaTeX math formulas using double-dollar `$$...$$` syntax parsed by `remark-math`. A formula embedded in prose renders inline; a formula in its own paragraph renders as display math. Single-dollar math is disabled so ordinary prose with dollar amounts remains readable. KaTeX `\gdef` macro definitions apply to following formulas in the same note only. Optional render-only support is acceptable for fenced extensions such as Mermaid when they remain ordinary markdown text. This gives Daily Notes a practical portability baseline while still allowing useful rendering features from markdown libraries. Custom syntax should not be required for the core note format because it would weaken source preservation and outside-Jot readability.
