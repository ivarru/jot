# Use CommonMark and GFM for Daily Notes

Jot Markdown is CommonMark with GitHub Flavored Markdown extensions, with optional render-only support for fenced extensions such as Mermaid when they remain ordinary markdown text. This gives Daily Notes a practical portability baseline while still allowing useful rendering features from markdown libraries. Custom syntax should not be required for the core note format because it would weaken source preservation and outside-Jot readability.
